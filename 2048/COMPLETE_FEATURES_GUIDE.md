### **What Are Inquiries?**
Inquiries are **quick contact forms** for tenants to express interest in a property without creating an account or logging in.

### **How Inquiries Work:**

#### **For Tenants (No Login Required):**
```
1. Browse properties on main page
2. Click "Inquire" button
3. Fill simple form:
   - Name
   - Email
   - Phone (+263 format)
   - Message
4. Submit inquiry
5. Landlord receives notification
```

#### **For Landlords:**
```
1. Go to "Inquiries" tab
2. See all inquiries for your properties
3. View tenant contact information
4. Contact tenant via phone/email
5. Respond to inquiry
```
